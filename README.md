# GT2TimSuite
A modern, fully-featured tool that handles exporting, editing, and creating PS1 Bitmap files (.tim)
This program runs on Python and the Pillow Library.
<p align="center">
  <img src="https://github.com/Silentwarior112/GT2Timsuite/blob/main/pics/clut.PNG">
</p>

## Supported Formats:
- Standard PS1 Single-CLUT TIM Formats: Type 2 TrueColor, Type 8 4bpp, Type 9 8bpp
- Standard PS1 Multi-CLUT TIM Formats: Type 8 4bpp, Type 9 8bpp
- Nonstandard Gran Turismo 2 4bpp Texture Sheets
- Gran Turismo 2 Multi-file containers: TIM packages and generic file packages
- #### Standard, .PNG files used for source images to convert into TIM

## Features
- TIM viewer: Fully-featured view window with manual CLUT configurator. Draw boxes over the graphics and set the palette
<p align="center">
  <img src="https://github.com/Silentwarior112/GT2Timsuite/blob/main/pics/edit.png">
</p>

- Accurate Black-keying transparency reproduction
<p align="center">
  <img src="https://github.com/Silentwarior112/GT2Timsuite/blob/main/pics/transparency.PNG">
</p>

- Converters: Export TIM files to standard PNG image(s). Multi-CLUT TIMs can be exported with user-configured CLUT information to rebuild back to TIM
<p align="center">
  <img src="https://github.com/Silentwarior112/GT2Timsuite/blob/main/pics/pngProject.PNG">
</p>

- Builders: Create custom multi-CLUT TIM files: Nonstandard Gran Turismo 2 4bpp Texture Sheets, and standard 4bpp and 8bpp multi-CLUT TIM files.
