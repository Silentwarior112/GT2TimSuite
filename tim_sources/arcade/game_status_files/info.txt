When modifying the tachometers, be aware that all 10 of them must have the exact same
16-color palette.
The image "tachs_all" is a 5x2 layout of every tachometer backing to use for quantizing them all to the exact same
16 color palette.
When you are ready to quantize them all, replace all 10 in this image with your custom ones.
Quantize this single image to 16 colors, then transplant each 80x80 section into their individual png files.
Additionally transplant the first six 80x80 sections into the 00_tachs.png file inside the sheet folder.

When building:
Individual tach images: Use "PNG to Standard Tim". After it finishes, check the log making sure each one is 4bpp.
The sheet: Use "Build 4bpp GT2 Tim Sheet".

Place the sheet's .tim and all 10 tach .tim files into a single folder with nothing else inside.
Use "Build Multi-File" to package them, save as game_status_files with no file extension.
Open the game_status_files.gz archive (7zip works for this) and drag & drop the new
game_status_files file into the .gz archive, which will replace the old one.
Pack VOL and build iso, check work.

Also, for tachs 14k and 16k, their numbering lacks their outlines in the original dump.
This is irrelevant for custom texture uses. When making your own custom tachs, include the outline around the numbers for 14 and 16 as well.
